ğŸ“– Tentang MathMaster

MathMaster adalah aplikasi web edukasi open-source yang dirancang untuk membantu pelajar Indonesia memahami matematika dengan cara yang lebih menyenangkan dan terstruktur.

Berbeda dengan metode belajar acak, MathMaster menggunakan konsep Learning Path (Peta Belajar) berbentuk tree structure yang memandu siswa dari materi paling dasar hingga mahir, disesuaikan dengan jenjang kelas (SD Kelas 1-6, SMP 7-9, SMA 10-12).

Dilengkapi dengan AI Tutor yang siap menjawab pertanyaan kapan saja secara gratis!

ğŸ› ï¸ Cara Menjalankan di Komputer (Local)

Ingin mengutak-atik kode atau menjalankannya di laptop sendiri? Ikuti langkah ini:

Prasyarat

Node.js (Versi 18 atau terbaru)

Git

Instalasi

Clone Repository

git clone [https://github.com/AhRoiz/MathMaster.git](https://github.com/AhRoiz/MathMaster.git)
cd MathMaster


Install Dependencies

npm install


Setup Environment Variables
Buat file .env di folder root, lalu isi API Key Google Gemini kamu:

VITE_GEMINI_API_KEY=Isi_Dengan_API_Key_Gemini_Kamu


Dapatkan API Key gratis di Google AI Studio

Jalankan Project

npm run dev


Buka http://localhost:5173 di browser kamu.
